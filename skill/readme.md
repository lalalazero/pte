# 79 分练习目标

## 重点题型

分值占比高，难度大

### 听力

- WFD 每天50个精听
- HIW 每天10 篇 HIW，1.2 倍速到1.5倍速，要一次性全对。
- FIB 练习 WFD 精听就够，精听 WFD 的时候会同时训练到 FIB 的能力。

### 阅读

- FIB(r) 每天15个精读
- FIB(r & w) 每天15个精读

### 口语

- RA 每天1篇同步跟读，3篇自选RA
- RS 每天30句
- DI 每天10句
- RL 每天2-3个
